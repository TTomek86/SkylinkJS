URLBuilderFn

( options ,   fn ,   )
private
  Available since 0.7.0

Function that fetchs from HTTP URL (build from options parameter provided) to start connection session to.

Parameters:

  • options {JSON}

    The parameter options. See URLBuilderFn() function for full parameter options

  • fn {Function}

    The parameter callback function that triggers on HTTP request state change.

    • state {String}

      The callback function parameter payload to determine the current state of the HTTP request.

      "loading": The state when loading HTTP request.
      "loaded": The state when connection session has started.
      "error": The state when connection session has errors
      "retry": The state when retrying loading HTTP request when timeout. Maximum retries are 5

    • data {JSON}

      The callback function parameter payload that contains the error object which follows the readyStateChange error event payload or the connection session data.
